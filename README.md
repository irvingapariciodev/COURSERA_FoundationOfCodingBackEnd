# ﻿🚀 Async Programming & C# Practice

Hands-on C# console exercises focused on coding Back-End C# principles. Each menu option represents a small, isolated scenario you can run, modify, and extend.

# 🧭 What’s Inside

This project uses a menu-driven console application to demonstrate Back-End C# principles such as variables, constant, POO.

# 📋 Menu Overview

1️ -Our first Hello World! (basic console output)

2️ - Variables typology and basic operations (integers, strings, booleans)

3️ - Display variables and constants in action

4️ - Create a basic ToDo List application (add, remove, display tasks)

5️ - Start with POO concepts: create a simple class (e.g., Animal) with properties and methods

6️ - Apply POO concept: Polymorphism with a base class and derived classes

7️ - Apply POO concept: Inheritance with a base class and derived

8️ - Keep practice with POO: Creating more objects

9️ - Create a simple console-based system grade management system for students (add students, assign grades, calculate averages)

10 - Exit the application

# 🎯 Learning Goals

•	Learn how to create and run a C# console application

•	Understand basic C# syntax and concepts (variables, constants, data types)

•	Practice structuring a console application with a menu system

•	Work with multiple objects and classes to apply POO principles

•	Implement basic user input validation and control flow
